# Запуск бота на Windows

Это краткое руководство по запуску Telegram-бота на Windows.

## Требования

- **Python 3.8+** установлен на вашем компьютере
  - Скачать можно с [python.org](https://www.python.org/downloads/)
  - При установке **обязательно** отметьте галочку "Add Python to PATH"

## Быстрый старт

### Вариант 1: Автоматический запуск (Рекомендуется)

1. **Откройте папку с ботом** в Проводнике Windows
2. **Дважды кликните** на файл `run.bat`
3. Готово! Скрипт автоматически:
   - Создаст виртуальное окружение (при первом запуске)
   - Установит все зависимости
   - Запустит бота

### Вариант 2: Ручной запуск

Если вы предпочитаете больший контроль, выполните следующие шаги:

1. **Откройте Command Prompt (cmd)** или **PowerShell**
2. **Перейдите в папку с ботом:**
   ```bash
   cd "C:\путь\к\папке\prod bot"
   ```

3. **Создайте виртуальное окружение** (только при первом запуске):
   ```bash
   python -m venv venv
   ```

4. **Активируйте виртуальное окружение:**
   ```bash
   venv\Scripts\activate
   ```
   После активации вы увидите `(venv)` в начале строки

5. **Установите зависимости** (только при первом запуске):
   ```bash
   pip install -r requirements.txt
   ```

6. **Запустите бота:**
   ```bash
   python run_modern_bot.py
   ```

## Настройка

Перед первым запуском убедитесь, что файл `.env` настроен правильно:

```env
BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_USER_ID=ваш_telegram_id
```

Если файла `.env` нет, создайте его на основе `.env.example`:
```bash
copy .env.example .env
```

Затем откройте `.env` в текстовом редакторе (например, Notepad) и заполните необходимые данные.

## Остановка бота

- Нажмите `Ctrl+C` в окне терминала

## Возможные проблемы

### "Python is not recognized"
**Решение:** Python не добавлен в PATH. Переустановите Python и отметьте галочку "Add Python to PATH".

### "pip is not recognized"
**Решение:** Обновите pip:
```bash
python -m pip install --upgrade pip
```

### Ошибка "ModuleNotFoundError"
**Решение:** Убедитесь, что виртуальное окружение активировано и зависимости установлены:
```bash
venv\Scripts\activate
pip install -r requirements.txt
```

### Бот не запускается
**Решение:** Проверьте:
1. Правильность токена в `.env`
2. Наличие интернет-соединения
3. Логи в папке `logs/`

### Проверка статуса бота
Если бот уже запущен, попытка запустить его снова покажет сообщение:
```
❌ ОШИБКА: Бот уже запущен (PID XXXXX)
```

## Автозапуск при включении компьютера

Чтобы бот запускался автоматически при включении Windows:

1. Нажмите `Win+R` и введите `shell:startup`
2. Создайте ярлык `run.bat` в открывшейся папке
3. Готово! Бот будет запускаться при входе в Windows
