## ⚡ Быстрый старт

### Вариант 1: Архив для переноса

```bash
# Создать архив
cd /Users/oleg/Project_SKLAD
tar -czf bot_deploy.tar.gz docker_deploy/

# На другом компьютере распаковать и запустить
tar -xzf bot_deploy.tar.gz
cd docker_deploy
docker-compose up -d
```

### Вариант 2: Скопировать папку

```bash
# Просто скопируйте всю папку docker_deploy на другой компьютер
# Затем:
cd docker_deploy
docker-compose up -d
```

## ⚠️ ВАЖНО перед запуском на другом компьютере

1. ✅ **Проверьте файл `.env`** - он должен содержать правильные токены
2. ✅ **Установите Docker** на новом компьютере
3. ✅ **Освободите порт 8080** или измените его в `docker-compose.yml`
