# Документация к экономической модели склада

## Основные параметры склада
*   **Общая площадь (м²)**: Общая арендуемая площадь склада в квадратных метрах.
*   **Стоимость аренды (руб./м²/мес.)**: Ежемесячная арендная плата за один квадратный метр.
*   **Доля полезной площади (%)**: Процент полезной площади от общей площади склада (за вычетом проходов, стен и т.д).

## Распределение площади
*   **Режим распределения площади**:
    *   **Ручной**: Позволяет задать площадь для каждого типа хранения вручную.
    *  **Автоматический**: Площади распределяются автоматически в соответствии с долями.
*   **Простое (м²)**: Площадь под простое хранение.
*   **Займы (м²)**: Площадь, используемая для хранения вещей под залог.
*   **VIP (м²)**: Площадь под VIP-хранение.
*   **Краткосрочное (м²)**: Площадь под краткосрочное хранение.

## Тарифы и плотности
*   **Тариф простого (руб./м²/мес.)**: Ежемесячный тариф за простой склад (руб./м²).
*   **Полок на 1 м²**: Количество полок на 1 м².
*   **Тариф краткосрочного (руб./день/м²)**: Тариф за 1 м² краткосрочного хранения в день.
*   **Наценка VIP (руб./м²/мес.)**: Дополнительная наценка для VIP-хранения (руб./м²).

## Оценка и займы
*   **Оценка вещи (%)**: Процент оценки стоимости вещи, которую можно взять под залог.
*   **Наценка реализации (%)**: Наценка, применяемая при продаже невостребованных вещей.
*   **Средняя оценка вещи (руб.)**: Средняя стоимость одной вещи (руб.).
*   **Ставка займов (%/день)**: Дневная процентная ставка для займов.
*   **Средний срок займа (дней)**: Средний срок займа в днях.

## Реализация (%)
*   **Реализация простое (%)**: Процент вещей из простого хранения, идущих на реализацию.
*   **Реализация займы (%)**: Процент вещей из займов, идущих на реализацию.
*   **Реализация VIP (%)**: Процент вещей из VIP-хранения, идущих на реализацию.
*   **Реализация краткосрочное (%)**: Процент вещей из краткосрочного хранения на реализацию.

## Процент заполняемости
*   **Заполнение простое (%)**: Процент заполнения площади простого хранения.
*  **Заполнение займы (%)**: Процент заполнения площади займов.
*   **Заполнение VIP (%)**: Процент заполнения VIP-секции.
*   **Заполнение краткосрочное (%)**: Процент заполнения краткосрочного хранения.

## Плотность (вещей/м²)
*   **Плотность простое (вещей/м²)**: Плотность хранения (вещей на м²) для простого хранения.
*  **Плотность займы (вещей/м²)**: Плотность хранения для займов (вещи/м²).
*   **Плотность VIP (вещей/м²)**: Плотность хранения для VIP (вещи/м²).
*   **Плотность краткосрочное (вещей/м²)**: Плотность хранения для краткосрочного (вещи/м²).

## Отток клиентов/вещей (%)
*   **Отток простое (%)**: Ежемесячный отток клиентов из простого хранения.
*  **Отток займы (%)**: Ежемесячный отток по займам.
*   **Отток VIP (%)**: Ежемесячный отток клиентов из VIP-секции.
*   **Отток краткосрочное (%)**: Ежемесячный отток клиентов по краткосрочному хранению.

## Финансы (ежемесячные)
*   **Зарплата (руб./мес.)**: Общие затраты на зарплату (руб./мес.).
*   **Прочие (руб./мес.)**: Прочие ежемесячные расходы.
*   **Амортизация (руб./мес.)**: Ежемесячная амортизация.
*   **Маркетинг (руб./мес.)**: Затраты на маркетинг.
*  **Страхование (руб./мес.)**: Ежемесячная страховка.
*  **Налоги (руб./мес.)**: Налоговые отчисления (руб./мес).
*   **Коммуналка (руб./мес.)**: Коммунальные услуги (руб./мес.).
*   **Обслуживание (руб./мес.)**: Обслуживание склада (руб./мес).

## Финансы (единовременные)
*   **Настройка (руб.)**: Единовременные затраты на настройку склада.
*   **Оборудование (руб.)**: Единовременные затраты на оборудование.
*   **Другие (руб.)**: Прочие единовременные расходы.
*   **Юридические (руб.)**: Единовременные юридические расходы.
*   **Логистика (руб.)**: Единовременные логистические расходы.

## Переменные расходы
*  **Упаковка (руб./м²)**: Стоимость упаковки на 1 м² площади.
*   **Электричество (руб./м²)**: Стоимость электроэнергии на 1 м².

## Инфляция и рост
*   **Инфляция (%/мес.)**: Ежемесячная инфляция (%).
*   **Рост аренды (%/мес.)**: Рост аренды в месяц (%).
*   **Рост зарплаты (%/мес.)**: Ежемесячный рост зарплаты (%).
*   **Рост прочих расходов (%/мес.)**: Ежемесячный рост прочих расходов (%).

## Расширенные параметры
*   **Отключить расширенные параметры**: Если включено, расширенные параметры игнорируются.
*   **Горизонт прогноза (мес.)**: Сколько месяцев прогнозируем.
*   **Вероятность невозврата (%)**: Вероятность невозврата (для займов).
*   **Коэффициент ликвидности**: Коэффициент ликвидности.
*   **Коэффициент запаса**: Коэффициент запаса для устойчивости.
*  **Льготный период (мес.)**: Льготный период по займам (мес).
*   **Рост доходов (%/мес.)**: Предполагаемый рост доходов в месяц.
*   **Рост расходов (%/мес.)**: Предполагаемый рост расходов в месяц.
*  **Метод прогнозирования**: Метод, используемый для построения прогноза.
*   **Симуляций Монте-Карло**: Число симуляций в Монте-Карло.
*   **Отклонения (0.1 = ±10%)**: Отклонение для Монте-Карло (доля).
*   **Seed**: Зерно случайности (Монте-Карло).
 *  **Распределение**: Тип распределения для симуляции Монте-Карло.
  * **Среднее (Норм. распр.)**: Среднее для нормального распределения.
  * **Ст. отклонение (Норм. распр.)**: Ст. отклонение для нормального распределения.
   * **Мин. значение (Треуг. распр.)**: Мин. значение для треугольного распределения.
   * **Мода (Треуг. распр.)**: Мода для треугольного распределения.
    * **Макс. значение (Треуг. распр.)**: Макс. значение для треугольного распределения.
*   **Включить расширенный ML-прогноз**: Включает дополнительные настройки для ML-прогноза.
*   **Степень полинома**: Степень полинома для полиномиальной регрессии.
*  **Количество деревьев**: Количество деревьев для случайного леса.
*  **Признаки**: Признаки для обучения ML-модели.
* **Поиск параметров**: Метод поиска параметров ML моделей.

## Основные термины:

*   **Прибыль**: Доходы - расходы.
*   **Маржа прибыли**: (Прибыль / Доходы) * 100
*   **Рентабельность**: (Прибыль / Расходы) * 100
*  **ROI (Return on Investment)**: ((Доходы - Расходы) / Расходы) * 100, показывает возврат на инвестиции.
*   **IRR (Internal Rate of Return)**: Внутренняя норма доходности (показывает доходность проекта с учётом дисконтирования).
*   **BEP (Точка безубыточности)**: Минимальный уровень дохода, при котором склад не убыточен.

## Как использовать

1.  Используйте боковую панель для ввода параметров.
2.  Просматривайте результаты на соответствующих вкладках.
3.  Используйте возможности настройки UI (цвет, тема), и ML моделей.
